LargestEmpires


General

The project, "Largest Empires", engages in the subject of largest empires in the history of mankind.

The project provides information about the culture, economy, areas, accomplishments etc. of each one of the empires.

The site offer the opportunity to visualize, explore and learn about history in the context of empires all over the globe.

The site provides services of search by: name, year, and area size.

In additon, exporting data to multiple formats- json,html and excel, is possiable.


Challenges

  • Finding a data source which allows to export information efficiently.
  • Processing the data in the most abstruct and general way, in order to present as much information as possible.
  • Exporting the information from our project to various formats.

Progress

We chose the main empires according to Wikipedia, for each empire we imported the information, metadata and maps abstractly.

We used the service Wikipedia supplies in order to receive the data as comfortable jason format, therefore the data is dynamic. Meaning if the data is changed in Wikipedia so is ours.

After importing the information we needed to execute parsing of the data, removing hyperlinks, extract the metadata and maps.

In addition we added the option to export the metadata of each empire in Excel, HTML and JSON.

Farther more we added 3 search options in our site: by name, year and minimum area size.

Digital Humanities Context

The project issues in finding, represntation of data in the sub-subject of digital humanities, history of human-kind.
It also allows searching the data in many ways, which is also a part of a Digital Humanities project.
